vite.config.ts

路径别名

  • replacement 后面得手动拼接上 “/”
// /@/views/ 默认转成 ./srcviews
export default {
  resolve: {
    alias: [
      {
        find: /^\/@\//,
        replacement: `${resolve(__dirname, "./src")}/`,
      },
    ],
  },
};
  • tsconfig.json 配置 paths 时 必须加上 baseUrl
{
  "baseUrl": ".",
  "paths": {
    "/@/*": ["src/*"]
  }
}

环境变量

想在文件中使用 import.meta.env 读取一些环境变量，需要在 tsconfig.json 中加入

{
  "types": ["vite/client"]
}
