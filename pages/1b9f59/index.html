  1. 监听插槽的变化

MutationObserver 接口提供了监视对 DOM 树所做更改的能力。它被设计为旧的 Mutation Events 功能的替代品，该功能是 DOM3 Events 规范的一部分。

export default {
  mounted() {
    // 当有变化时调用`update`
    const observer = new MutationObserver(this.update);

    // 监听此组件的变化
    observer.observe(this.$el, {
      childList: true,
      subtree: true,
    });
  },
};

# 递归插槽

<!-- VFor.vue -->
<template>
  <div>
    <!--  渲染第一项 -->
    {{ list[0] }}
    <!-- 如果我们有更多的项目，继续!但是不要使用我们刚刚渲染的项 -->
    <v-for v-if="list.length > 1" :list="list.slice(1)" />
  </div>
</template>
  • 使用插槽，设置默认内容
  • 方便对特殊的子项进行更改
<template>
  <div>
    <!-- Pass the item into the slot to be rendered -->
    <slot v-bind:item="list[0]">
      <!-- Default -->
      {{ list[0] }}
    </slot>

    <v-for v-if="list.length > 1" :list="list.slice(1)">
      <!-- Recursively pass down scoped slot -->
      <template v-slot="{ item }">
        <slot v-bind:item="item" />
      </template>
    </v-for>
  </div>
</template>
  • 使用
<template>
  <div>
    <!-- 常规列表 -->
    <v-for :list="list" />

    <!-- 加粗的项目列表 -->
    <v-for :list="list">
      <template v-slot="{ item }">
        <strong>{{ item }}</strong>
      </template>
    </v-for>
  </div>
</template>

# Suspense

  • Suspense 的 template 下只能一个子元素 DOM，多了显示不出来
  • 不过 antd 给的提示说 api 可能会变
<!-- 错误写法 -->
<!-- <Suspense> is an experimental feature and its API will likely change. -->
<Suspense>
  <template #default>
    <div></div>
    <div></div>
  </template>
</Suspense>

<!-- 正确写法 -->
<Suspense>
  <template #default>
    <div>
      <div></div>
      <div></div>
    </div>
  </template>
</Suspense>

# 自定义组件

  • 不需要对自组件进行绑定也可以触发事件
  • 给 Select 组件绑定的 v-model:value 和@change 都不要代理
  • 当 Select 组件的根组件直接是 a-select 时，可以不需要代理，属性将默认传入 a-select 组件
<!-- Parent.vue -->
<template>
  <Select
    v-model:value="state.value"
    :options="state.options"
    @change="changeValue"
  ></Select>
</template>

<!-- Select.vue -->
<template>
  <a-select ref="select">
    <a-select-option v-for="(item, i) in options" :key="i" :value="item.value">
      {{ item.label }}
    </a-select-option>
  </a-select>
</template>

# 一些特性

# Attribute 继承

props 和 emits property 中未包含的所有属性将会自动绑定到自组件的根节点上

  • 可以使用 inheritAttrs: false 禁用
  • 多个根节点组件不会自动绑定，将会发出警告，使用 v-bind 绑定后则清楚警告

# teleport

将一个 DOM 挂载到另一个 DOM 上，多个 <teleport> 组件可以将其内容挂载到同一个目标元素

<teleport to="#modals">
  <div>A</div>
</teleport>
<teleport to="#modals">
  <div>B</div>
</teleport>
<!-- result-->
<div id="modals">
  <div>A</div>
  <div>B</div>
</div>
