# Introduction

## 介绍

这是一个使用 gitbook 构建的学习文档，包含`vue` `react` `node` `vite` `git` `html` `http` `axios` `javascript`等等的前端相关技术的学习，其中的内容大部分是我工作中会遇到的、面试中遇到的、或者一些想要了解的知识点，本项目会长时间不间断的持续更新。

## Install

- download

```
git clone https://github.com/shellingfordly/web-doc.git
```

- run

```
gitbook serve
```
